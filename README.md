# はじめに
本サイトは[（ソフトウェア品質技術者のための）データ分析勉強会][1]のサイトに収まらなくなった情報等を整理するためのGitHubリポジトリです。

## サイトの構成
本ページは以下のコンテンツから構成されます。

### Archives
過去の[データ分析勉強会][1]で利用された資料の保管庫です。順次、保管先にリンクを張っていきます。

### Cabinet
[データ分析勉強会のサイト][1]のCabinet, Tipsの保管庫です。別リポジトリとなります。

# Cabinet
[データ分析勉強会のサイト][1]にあるCabinetならびにTipsを整理しGitHubリポジトリです。整理するにあたり分類の基準はRStudio社がGitHubで公開している[RStartHere][3]で説明されているData Science Workflowの各プロセスを参考にしています。

## Program
データ分析に関わるプログラミング、プログラミング環境、ツール類に関する事項を扱います。

## Tidy
データハンドリングの観点からTransformへ集約します。

## Transform
データハンドリングに関する事項を扱います。

## Visualize
可視化（主にグラフ描画）に関する事項を扱います。

## Model
[Model](https://k-metrics.github.io/model/)は[データ分析勉強会][1]のメインである分析手法や例題・演習問題の回答例に関する事項を扱います。

## Infer
Modelへ集約します。

## Communicate
[Communicate](https://k-metrics.github.io//communicate)は分析結果の文書化（主にR Markdown）に関する事項を扱います。

## Automate
当面、本分類の対象となる資料はありません。

# Archive
過去の勉強会の講義資料置き場です。

# Links
統計分析やソフトウェアメトリクスに関わる情報へのリンク集です。
